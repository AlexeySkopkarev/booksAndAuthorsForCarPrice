Задание:
Реализовать REST API каталога книг

Модели:
- Авторы (ФИО, год рождения, год смерти (если есть))
- Книги (Название, год публикации, краткое содержание, связь с авторами)

Требования:
- Реализовать приложение с использованием фреймворка Laravel
- Для каждой модели должен быть CRUD с поиском и пагинацией
- Должна быть возможность поиска книг по автору и автора по книге
- Вывод авторов должен быть с количеством его книг
- В выводе книг должен присутствовать список авторов

Будет плюсом:
- Наполнение тестовыми данными
- Наличие описания API по спецификации OpenAPI (swagger)
- Реализовать аналогичное приложение на GO (каких-либо ограничений по используемым фреймворкам и библиотекам нет)
- Dockerfile (для php и go отдельно)
- Готовый docker-compose.yaml для быстрого старта проекта локально
